# curso-git

## 1. Iniciando o repositório
_git init_

## 2. Adicionando o arquivo README.md
_git add README.md_

## 3. Fazendo o commit
_git commit -m "Criando o arquivo README.md"_

## 4. Adicionando ao repositório remoto
_git remote add origin https://github.com/vinialbano/curso-git.git_

## 5. Fazendo o push
_git push -u origin master_
